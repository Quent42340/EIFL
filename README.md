# EIFL
## Epitech Intra for Linux

This project aims to provide a Linux interface to Epitech intranet.

### Requirements

For ArchLinux users:

- `cmake`
- `qt5-base`
- `qtkeychain`
- `curl`
- `cpr-git` (AUR)

### Requirements (for old python project)

- `python`
- `python-pyqt5`
- `python-keyring`
- `python-requests`
